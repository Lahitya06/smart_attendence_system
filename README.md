# Smart Student Attendance System

## Description
This project is a simple web-based student attendance system developed using **HTML, CSS, and JavaScript**. It allows teachers to mark student attendance, store records, and view attendance percentages.

## Features
- Display student list
- Mark students as Present or Absent
- Save attendance by date
- Calculate attendance percentage
- Store data using LocalStorage

## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Open **index.html** in your browser.
4. Select a date, mark attendance, and save.

## Project Files
- `index.html` – Main webpage
- `style.css` – Styling of the webpage
- `script.js` – JavaScript functionality
